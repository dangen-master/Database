Задание проекта
Проект: приложение для поиска вуза
Цель проекта: создать удобное приложение для отбора вузов для поступления
При запуски программы появляеются:
1)Таблица для вывода результатов запроса 
2)фильтр по вузам
3)Фильтр по экзаменам
4)фильтр бюджетных мест (все вузы\только те в которых есть бюджет)
5) кнопки для применения и сброса фильтров
6)Кнопка выхода
7)кнопка для пересоздания таблиц (Если не будут найдены нужные bd или пользователь захотеч обновить их)
